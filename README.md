# Micro-manage

CLI tool to manage your microserices architecture

# Install
Install using:
```bash
#npm:
npm install -g @the-1aw/micro-manage

#yarn:
yarn global add @the-1aw/micro-manage
```

Then you can either use:
```bash
#LONG VERSION:
  micro-manage ...args
#or SHORT VERSION:
  mm ...args
```

# Configuration

To run micro-manage configuration prompt use:
```bash
  micro-manage init
```
